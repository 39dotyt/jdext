## Installation

You should have [Node.JS and npm](https://nodejs.org/) installed.

```bash
$ npm intall -g jdext
```

## Usage

```bash
$ jdext C:\Users\<username>\Documents\JustDance2017\<id>\JDSave_1 myvideo.webm
```
